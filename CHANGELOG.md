# Changelog

## v0.1.0

### Added

- macOS menu bar entry.
- Single-main-window navigation.
- Custom workflows with editable actions and run results.
- Development environment checker.
- Command center for built-in safe commands.
- Downloads organizer.
- Settings page.
- App icon and menu bar icon assets.
- Release build, ZIP packaging, and DMG packaging scripts.

### Known Limitations

- The app may not be signed with an Apple Developer ID or notarized.
- Some commands depend on tools installed on the user's Mac.
- Docker CLI availability does not guarantee Docker Desktop is running.
- Some workflow actions require user configuration before they are useful.
- AI features are not connected yet.
